Setting up your app

Create your first app

There are two ways for you to create your first Errai application. You can use our sample app, Errai Tutorial, as a starting point, or you can create a barebones Errai app using JBoss Forge, that will have all the basic dependencies in place for you. Both options are explained in detail below. To get familiar with Errai, we recommend creating a new app and developing it using the steps in this guide. If you prefer to learn by example, the Errai Tutorial is a better option for you.

Errai Tutorial

The Errai Tutorial is a basic sample application that showcases all the major features of Errai. You can use it both as an example to explore Errai's functionality and as a base upon which to build your own more complex web app. The Errai Tutorial can be downloaded from the GitHub repository here.

JBoss Forge

To get you started with your first Errai app, we will use JBoss Forge and the Errai Forge Addon to create a new app with the basic dependencies already in place. Instructions for downloading and using the Forge addon are on the Errai Forge Addon Page.

Hello World!

Now that you have an empty Errai app set up with the Forge addon, you need to be able to run it. We'll first add some simple code so you can see something when you run the app. Make sure you have a copy of Wildfly 8 in the target/ directory of your new app for debugging purposes.

TIP: Before you add the code snippet below, use the Errai Forge Addon to add Errai IOC to your project so that your app runs properly.

Create a new subfolder client/local/ under the folder which contains your GWT module file (with the extension .gwt.xml). In our example, the GWT module file ErraiDemo.gwt.xml is located in the directory errai-forge-demo/src/main/java/org/jboss/errai/demo/.

In the new client/local/ subfolder, create a file called App.java with the following contents:

App.java
    package org.jboss.errai.demo.client.local;

    import javax.annotation.PostConstruct;
    import org.jboss.errai.ioc.client.api.EntryPoint;
    import com.google.gwt.user.client.Window;

    @EntryPoint
    public class App {
      @PostConstruct
      public void onLoad() {
        Window.alert("Hello World!");
      }
    }
